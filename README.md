# Packet Inspection Firewall Simulator

A C++ based firewall simulation system designed to demonstrate packet inspection, rule-based filtering, protocol segmentation, threat prioritization, and blacklist management using fundamental data structures.

## Features

- Packet Creation and Processing
- Rule-Based Firewall Engine
- Protocol Segmentation
- Threat Prioritization Queue
- Packet Trace Logging
- Dynamic Blacklist Management
- Firewall Decision Tracking

## Technologies Used

- C++
- Linked Lists
- Circular Linked List
- Queue
- Priority Queue
- Stack
- Dynamic Memory Allocation

## Learning Outcomes

This project was developed to strengthen understanding of:

- Data Structures and Algorithms
- Network Security Concepts
- Firewall Rule Processing
- Packet Inspection Techniques
- Secure System Design

- Persistent Rule Storage
